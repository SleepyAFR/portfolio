Cybersecurity Incident Reporting System

Developed a comprehensive, database-driven system for efficiently logging, tracking, and managing cybersecurity incidents. The system allows users to report incidents, specifying details like description, severity (low, medium, high), and the affected system (e.g., email server, payroll system). Admin users can manage and view all incidents, while non-admin users can submit new reports. Key features include automated daily and monthly reports, real-time incident trend analysis, and secure audit logging to ensure accountability.
